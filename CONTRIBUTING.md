# Contributing to FetchIt

Thank you for your interest in improving FetchIt! We welcome all contributions.

## How to Contribute

- **Report Bugs**: Open an issue if you find a problem.
- **Suggest Features**: Tell us how we can make FetchIt better.
- **Submit Code**: Fix a bug or add a feature.

## Development Setup

Follow these steps to set up your computer for development.

### 1. Download the Code

Use this command to download the code to your machine:
```bash
git clone https://github.com/Aarav90-cpu/fetchIt.git
```

### 2. Create a Virtual Environment

Use this command to create an isolated Python space:
```bash
python3 -m venv .venv
```

Use this command to activate the virtual environment:
```bash
source .venv/bin/activate
```

### 3. Install Dependencies

Use this command to install the project and its requirements:
```bash
pip install -e .
```

### 4. Compile the C++ Extension

Use this command to compile the C++ code for fast performance:
```bash
make
```

## Pull Request Process

Please follow this checklist before submitting your changes.

| Step | Action |
| --- | --- |
| 1 | Create a new branch for your feature. |
| 2 | Write clear and simple code. |
| 3 | Test your code locally to ensure it works. |
| 4 | Push your branch to GitHub. |
| 5 | Open a Pull Request and describe your changes. |

We will review your Pull Request and merge it once it is approved.
