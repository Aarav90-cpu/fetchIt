# Code of Conduct

We want everyone to feel welcome here. Please follow these rules.

## Our Rules

- **Be Respectful**: Treat everyone kindly.
- **Be Inclusive**: Welcome people of all backgrounds.
- **Be Helpful**: Offer guidance and support to others.
- **Be Professional**: Keep discussions focused on the project.

## Unacceptable Behavior

- Harassment of any kind is not allowed.
- Do not use offensive language.
- Do not post spam or unrelated content.

## Reporting Issues

If you see someone breaking these rules, please report them.
- Email the project maintainers.
- Open an issue on GitHub if appropriate.

We will review all reports and take action when necessary.
